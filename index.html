Object Oriented Programming

OOP stands for Object-Oriented Programming. It is a programming style that views a program as a collection of objects, where each object is an instance of a class. These classes are based on concepts such as Inheritance, Composition, and Polymorphism, allowing programmers to effectively organize and structure code and provide a more organized structure.
